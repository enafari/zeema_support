📋 Available APIs

📊 Get Invested Plans API

🔍 Description

Retrieves invested plans information from the Supabase database based on national ID.

Endpoint: invested_plans_by_users

Filter: national_id

Returns: User information with associated plans and transactions

📋 Get Plan Phase API

🔍 Description

Retrieves plan phases from the Supabase plan_phase table based on plan ID.

Endpoint: plan_phase

Filter: plan_id

Returns: Array of plan phases matching the plan ID

📅 Get Plan Phase Timeline API

🔍 Description

Retrieves plan phase timeline data from the Supabase plan_phase_timeline table based on a list of plan IDs.

Endpoint: plan_phase_timeline

Filter: plan_id (multiple values supported)

Returns: Array of plan phase timeline data matching the provided plan IDs

💬 Insert Chat ID API

🔍 Description

Generates a new unique chat_id and inserts it into the Supabase chats table.

Endpoint: chats

Method: POST

Returns: New chat_id and created_at timestamp

👤 Insert National ID API

🔍 Description

Inserts national_id into the chats table based on chat_id.

Endpoint: chats

Method: POST

Parameters: national_id, chat_id

Returns: Updated chat information with national_id

🔍 Get Chat by ID API

🔍 Description

Retrieves chat information from the Supabase chats table based on chat_id.

Endpoint: chats

Filter: chat_id

Returns: Chat information including chat_id, user_id, and created_at
